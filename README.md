## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ivan-tarasov/fh-trip-builder.git
   ```
2. Open site in browser
3. Enter your MySQL credentials in `config.mysql.php`
   ```php
   $___mysql = [
     'host'  => 'localhost',
     'user'  => 'dbuser',
     'pass'  => 'dbpass',
     'db'    => 'dbname'
   ];
   ```
